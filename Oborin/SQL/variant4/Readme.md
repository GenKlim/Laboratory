4.	Приложение для торгового предприятия должно обеспечить учет информации о своих сотрудниках. О сотрудниках должна храниться информация: фамилия, имя, адрес, дата рождения, но-мера телефонов, дата приема на работу, название занимаемой должности (управляющий, помощник управляющего, продавец на полный рабочий день, продавец на неполный рабочий день). Для должности может потребоваться краткое описание обязанностей, возможно только для некоторых. Каждый сотрудник может зани-мать только одну должность.
